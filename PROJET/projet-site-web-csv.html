Projets - Site Web

Ce projet vise à concevoir un site web permettant d’importer, analyser et afficher dynamiquement des données issues d’un fichier CSV. Il met l’accent sur la structuration des informations, leur lisibilité et une interface ergonomique afin de faciliter la compréhension et l’exploitation des données.

En cours ...

Objectifs

✔ Importer et afficher un fichier CSV

✔ Structurer les données dans une interface web

✔ Améliorer la lisibilité et l’ergonomie

✔ Manipuler des données via Python
